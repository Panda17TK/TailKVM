# TailKVM v0.1.0 — Bob-note verification build #5（使い方改善）

bobnote-4 の実機フィードバックに対応した**UX 改善版**。機能は同じだが「接続はできるがマウスが動かない／
モニターやボタンの意味が分からない」を解消。

## 主な変更

- **クイックスタートのカードを最上部に追加**:
  - ① 相手PCの Tailscale IP を入れて「接続」
  - ② 「マウス共有を開始（ミラー）」を押すと**すぐにマウスが相手PCで動く**（画面端に当てる操作は不要）
  - 「停止」で自分の操作に戻る
  - **このPCのモニター構成を常時、縮尺表示**（解像度・スケール%・主モニタ・負座標の有無）
- 旧来は「接続しただけでは何も転送されない」「リモートモードは画面端に当てないと作動しない」「モニター図は
  ピア選択後しか出ない」ため動かないように見えていた問題を、クイックスタートで回避。
- 用語・見出しの明確化、ヒーローの古い表記を修正。

> 高度な設定（リモートモード/シームレス絶対モード/マルチスクリーン配置 2D エディタ/クリップボード等）は
> 下部のカードに従来どおりあります。

## 使い方（最短）

1. 両方の PC で TailKVM を起動。
2. **操作される側（例: 他PC）**: 下部「TCP Session」で `Start receiver` を押す。
3. **操作する側（メインPC）**: 最上部クイックスタートに相手の Tailscale IP（`100.x.y.z`）を入れて「① 接続」。
4. 「② マウス共有を開始（ミラー）」を押し、マウスを動かす → 相手のカーソルが動く。
5. 「停止」で戻る。
